# Spam Detection App# Spam Detection App
# Spam Detection Dashboard

A Machine Learning based Spam Detection Web Application built using:

- React.js
- Flask
- Scikit-Learn
- Logistic Regression
- TF-IDF

---

## Features

- Spam Detection
- Spam Probability Score
- Model Accuracy Display
- Dark Mode
- Prediction History
- Responsive Design
- Local Storage Support

---

## Tech Stack

Frontend:
- React
- Axios
- CSS

Backend:
- Flask
- Pandas
- NumPy
- Scikit-Learn

Machine Learning:
- Logistic Regression
- TF-IDF Vectorization
